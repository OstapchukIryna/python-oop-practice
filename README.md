# Python OOP Practice

Object-oriented Python exercises: designing classes with well-thought-out
interfaces, appropriate use of dunder methods, encapsulation, and clean APIs.

Each example is a small self-contained project focused on a specific
design concept.

## Contents

- [Inventory Item](./inventory_item/) — arithmetic operations, comparison,
  and string representation for a domain object.

## Approach

For each example:
- Class code with type hints and docstrings
- README describing the design decisions and semantics
- Unit tests covering behavior and edge cases
